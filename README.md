# Birthday Cake 🎂

생일 축하 케이크 웹사이트입니다!

## 기능

- 🎂 3D 케이크 애니메이션
- 🎨 3가지 테마 (초콜릿, 크림, 블루베리)
- ✏️ 이름과 메시지 편집
- 🎉 컨페티 효과
- 📱 반응형 디자인
- 🔗 URL 공유 기능

## 배포 상태

✅ GitHub Pages에 성공적으로 배포되었습니다!
🌐 사이트: https://hyyrim.github.io/birthday-cake

## 개발

```bash
# 의존성 설치
pnpm install

# 개발 서버 실행
pnpm dev

# 빌드
pnpm build
```

## 기술 스택

- React 19
- TypeScript
- Vite
- CSS Modules
- Canvas Confetti
